PokeRateHub

ポケモングローバルリンク(PGL) から対戦履歴を取得し、GitHubのHeatmapのような可視化をする自作Webアプリです。 このツールを使うと上のスクリーンショットのようなグラフを表示します。

基本的にはNode.jsを使用していて、フレームワークにはVue.jsを使用しています。 一部Pythonも使用しています。

ポケモングローバルリンク のサービスが2020年2月に終了したため，本リポジトリは動作しません．
Go to repository

Atam

競技プログラミングサイト「 AtCoder 」へのソースコードの提出を、 ブラウザを介さずターミナルから提出できるようにするツールです。 Node.jsを使用しています。

僕の所属しているサークルで、チームを組んで共同開発しました。 僕はログイン処理などを実装しました。

Go to repository
